# Simple Interest Calculator

## Description
This project contains a simple Bash script that calculates **Simple Interest** based on user input.

The calculator takes the following inputs from the user:
- Principal amount
- Rate of interest
- Time period

Using these values, it calculates the **Simple Interest** using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Features
- Takes input from the user
- Calculates simple interest
- Displays the result in the terminal
- Written using Bash shell scripting

## Files in this Repository
- `simple-interest.sh` – Bash script to calculate simple interest
- `README.md` – Project documentation

## Usage
1. Run the script in the terminal.
2. Enter the principal amount.
3. Enter the rate of interest.
4. Enter the time period.
5. The script will calculate and display the simple interest.
